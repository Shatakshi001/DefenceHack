HTML Nested Lists

A nested list in HTML is a list that contains other lists within its list items. This creates a hierarchical structure, where each sublist is indented to visually represent its relationship to the parent list item.

  1. Nested Unordered List

    An unordered list in HTML is a collection of items represented by bullet points, providing a flexible way to display information without a specific order. Nested Unordered List is when an Unordered List is present inside an Unordered List.

  2. Nested Ordered List

    In HTML, an ordered list organizes items in a numbered sequence, providing a structured way to present information. Nested Ordered List is when an Ordered List is present inside an Ordered List.

  3. Nested Mixed(Unordered+Ordered) List

    In HTML, a mixed nested list organizes data in random format. Here an ordered list can be inside in an unordered list or vice versa.

  4. List with Headings

    In HTML, you can also use the heading tag inside the list tag.

Tables in HTML

A table in HTML consists of table cells inside rows and columns. An HTML table is created using the

tag. Inside the table, we use:
  • <tr>: Table Row. Each <tr> tag defines a row in the table.
  • <td>: Table Header. This tag is used for actual data cells under each column.
  • <th>: Table Data. It's used for the headers of the columns. Text in <th> tags is usually bold and centered by default.

Structure of a Table:

  1. <thead>: Table Header. Defines the header section of a table, often containing column labels.
  2. <tbody>: Table Body. Represents the main content area of a table, separating it from the header or footer.
  3. <tfoot>: Table Footer. Specifies the footer section of a table, typically holding summaries or totals.
